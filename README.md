# Varun Gurupurandar — Portfolio Website

Personal portfolio for AI/ML engineering work, hosted on GitHub Pages.

**Live site:** [gvarun20.github.io/MY_WEBSITE](https://gvarun20.github.io/MY_WEBSITE/)

## Stack

- HTML5, CSS3, Vanilla JavaScript
- GitHub Pages for hosting
- FormSubmit for contact form

## Local preview

Open `index.html` in a browser, or run a local server:

```bash
npx serve .
```

## Deploy

Push to the `master` branch — GitHub Pages serves automatically from this repo.
